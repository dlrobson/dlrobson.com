Line Following Robot

Built a line following robot from a provided PCB. Various parts had to be soldered on using through hole and surface mount soldering, including hall effect sensor, Schmidt triggers, and infrared LEDs. The microcontroller was programmed using C.

Created for the final project of MTE220 - Sensors and Instrumentation.
